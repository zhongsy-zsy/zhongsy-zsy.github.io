Hexo使用教程

[toc]

创建与发布文章

hexo new 文章名

hexo deploy -g 部署

删除文章

直接在source/_post文件目录下删除文章，再只想 hexo d -g

theme更改

首先将自己喜欢的主题下载到theme目录下。

有两个设置，一个在hexo主目录下面，配置使用哪一个主题(theme字段)，theme目录下面有具体的主题文件，当中含有一个配置文件用于配置细节。

安装依赖

在 hexo init目录下只想 nmp install xxx –save进行安装
